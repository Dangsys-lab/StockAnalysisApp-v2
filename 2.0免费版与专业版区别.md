## 五、数据获取方式差异

### 免费版

```
用户输入股票代码
    ↓
LocalIndicatorCalculator（本地计算5个基础指标）
    ↓
StockDataService → 新浪/腾讯 API（实时行情直连）
    ↓
展示结果（零服务器成本）
```

### 专业版

```
用户输入股票代码
    ↓
FC后端请求（?is_pro=true）
    ↓
IndicatorCalculatorV2（全量计算17个指标）
    ↓
返回完整指标数据 + 综合分析报告
    ↓
展示结果
```

## 六、购买与状态管理

### 内购流程

1. 用户点击"升级专业版"按钮 → 打开 PurchaseView
2. 点击"立即购买" → IAPManager 调用 StoreKit 发起购买
3. 购买成功 → `isProUser = true`，持久化到 UserDefaults
4. UI 自动切换到专业版界面

### 状态持久化

| Key | 类型 | 说明 |
|-----|------|------|
| `isProUser` | Bool | 是否专业版用户 |
| `purchaseDate` | Date | 购买日期 |

### 恢复购买

- 支持苹果"恢复购买"机制
- 调用 `IAPManager.restorePurchase()` 恢复已购状态

### 审核模式

- 检测 App Store 审核环境时自动解锁专业版功能
- 检测方式：`ForceReviewMode` UserDefaults / `STOREKIT_SANDBOX` 环境变量 / `sandboxReceipt`
- 可通过 `ProStatusManager.forceReviewMode()` / `disableReviewMode()` 手动控制

## 七、后端 API 区分

| API | 免费版参数 | 专业版参数 | 返回差异 |
|-----|-----------|-----------|----------|
| `GET /api/indicators/<code>` | `?is_pro=false` | `?is_pro=true` | 免费版仅返回趋势+摆动类；专业版返回全部4类 |
| `GET /api/indicators/list` | - | - | 同时返回 free_version 和 pro_version 列表 |
